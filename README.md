### to run 

Create the virtual environment  
`python -m venv venv`

Activate virtual environment  
```
venv\Scripts\activate.bat
./venv/Scripts/activate
source venv/bin/activate
```

Create a .env file  
```
AWSPROFILE=123456_Admin
REGION=eu-west-2
```

Run the app to call Bedrock  
`python callbedrock.py`
